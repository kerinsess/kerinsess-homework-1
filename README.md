
# ДЗ 1 — Лінійне обчислення числа π методом Монте-Карло (Java)

## Опис

Реалізація алгоритму обчислення числа π методом Монте-Карло на чистій Java (без сторонніх бібліотек).

Для кожного значення N фіксуються:
- оцінене значення π
- точність: `abs(π_est − π_true)`
- час виконання (секунди)
- метрика: `time_per_point = execution_time / N`

### Значення N, що тестуються:
| N |
|---|
| 1 000 000 |
| 10 000 000 |
| 100 000 000 |
| 1 000 000 000 |
| 10 000 000 000 |
| 100 000 000 000 |

## Алгоритм

Генеруємо випадкові точки `(x, y)` у квадраті `[0, 1] × [0, 1]`.  
Точка потрапляє в чверть кола, якщо `x² + y² ≤ 1`.

```
π ≈ 4 × (кількість точок всередині кола) / N
```

## Запуск

```bash
# Компіляція
javac PiMonteCarlo.java

# Запуск
java PiMonteCarlo
```

Результати зберігаються у файл:
```
results/pi_monte_carlo_results.csv
```

### Формат CSV:
```
N,pi_estimated,accuracy,execution_time_sec,time_per_point
```

## Відомі обмеження

- Використовується чистий лінійний цикл `for`, тому для великих N час виконання значний:
  - 1M → ~0.02 с
  - 1B → ~10–15 с
  - 100B → ~20–30 хв
- Результати є стохастичними — кожен запуск дає трохи інше значення π.
- Потрібна Java 11+.
